# ☕ Fundamentos de Java - Desafio 72h

Este repositório contém os exercícios práticos desenvolvidos durante o curso gratuito de lógica de programação da plataforma **Javanauta**. O objetivo foi consolidar a base da linguagem em um desafio intensivo de 3 dias.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Java 21 (LTS)
* **IDE:** IntelliJ IDEA Community Edition
* **JDK:** Eclipse Adoptium (OpenJDK)

## 📁 Estrutura do Projeto
As classes foram organizadas por temas conforme o cronograma da **Javanauta**:

* `EntradaESaidaDeDados.java`: Manipulação de `Scanner` e `System.out`.
* `EstruturaIfElse.java`: Lógica condicional e tomada de decisão.
* `SwitchCase.java`: Seleção múltipla de condições.
* `TrabalhandoComStrings.java`: Métodos como `charAt()`, `length()` e `substring()`.
* `EstruturasDeRepeticao.java`: Implementações de `for`, `while` e `do-while`.
* `ContadorDiasMudanca.java & CalculadorParcelas.java`: Desafios práticos autoral unindo os conceitos aprendidos.

## 🧠 Aprendizados
A imersão permitiu dominar a sintaxe do Java 21, entender precedência lógica em estruturas condicionais e praticar a resolução de problemas de forma independente (pausando as aulas e codando antes da solução).
